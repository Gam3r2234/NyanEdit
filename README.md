USE: 
Double click run.bat and choose a file, if no file is chosen, it will open defaultmap.mpd (empty room).
Click on the tile you want to place from the bottom and place it in the top area of the screen.
The last three buttons swap the forest, open a new file, and save respectivley.
Settings can be changed in settings.json.
File locations can be changed in settings.json and tiles.json.

INFO:
Levels are stored in Nyanfile\Map\ and are organized into 60 rows of 40 tiles.
Each tile is represented as 2 bytes.
Tiles 0xFFFB to 0xFFFF were added by me and shouldnt be replaced if adding new tiles.
Some tiles used in game have no texture and will display with the no texture tile, this is normal and intended.
